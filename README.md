# Shrawan Kumar Pathak Portfolio

Welcome to my personal portfolio website. This project showcases my background, skills, projects, education, and contact information in a modern and responsive design.

## About
This portfolio website is built using HTML, CSS, and JavaScript. It is designed to be clean, professional, and mobile-friendly so visitors can easily explore my work and connect with me.

## Features
- Responsive layout for desktop and mobile
- Modern UI with smooth scrolling
- Sections for Home, About, Skills, Projects, Education, and Contact
- Clean and professional design
- Easy to customize and deploy

## Technologies Used
- HTML5
- CSS3
- JavaScript
